## Setup
- Project environment:
    - [Node.js](https://nodejs.org/en/) installed
    - [MongoDB Shell](https://www.mongodb.com/docs/mongodb-shell/) installed


## Usage
- On project terminal
    - install dependencies `npm install`
    - start the app `npm app.js`
- On Mongodb Shell
    - start the database server `mongosh`
